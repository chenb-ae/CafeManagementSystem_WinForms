# ☕ Cafe Management System (Language: Vietnamese)
A desktop application for managing cafe operations, built using **C#** and **WinForms** with **SQL Server**.
## How to use
### 1. Database setup
1. Open **SQL Server Management Studio (SSMS)**.
2. Open `GamingCafe_script.sql` and execute the script to create the database, tables and sample datas.
### 2. Run the Application
1. Press **F5** or click the **Start** button in Visual Studio.
2. Default login:
   - 2.1. **Username:** admin
   - 2.2. **Password:** 123
